# KrateLabs REST Server

## Install

**Download** the latest GitHub repository.

```bash
$ git clone git@github.com:KrateLabs/KrateLabs-REST.git
$ cd KrateLabs-REST
$ npm install
```

Run as background process

```bash
$ npm run autostart
```

Open your favorite browser to `http://localhost:5000`
